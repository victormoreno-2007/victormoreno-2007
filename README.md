## 👨‍💻 Sobre mí

Desarrollador Junior orientado a la construcción de soluciones tecnológicas eficientes. Tengo experiencia práctica en todo el ciclo de vida del desarrollo de software (SDLC), desde el diseño de arquitecturas limpias y APIs RESTful, hasta la automatización de procesos operativos y el diseño de interfaces modernas y responsivas. 

Me apasiona el trabajo colaborativo bajo metodologías ágiles (Scrum) y aporto una mentalidad analítica, gran adaptabilidad y un compromiso absoluto con el aprendizaje continuo para generar un impacto positivo y escalable en cada proyecto.
